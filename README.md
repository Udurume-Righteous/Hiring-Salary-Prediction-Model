# Hiring Salary Prediction Model

## Project Overview
This project builds a Linear Regression model to help HR predict candidate salaries based on:

- Years of Experience
- Test Score
- Interview Score

## Steps Performed
- Data Cleaning (handled missing values and text conversion)
- Feature Selection
- Model Training using Linear Regression
- Salary Prediction for new candidates

## Predictions
- Candidate 1 (2 yrs, 9 test, 6 interview): ~53,290
- Candidate 2 (12 yrs, 10 test, 10 interview): ~92,268

## Tools Used
- Python
- Pandas
- Scikit-Learn
